One-Time Pad Cipher



Cryptanalysis:


A one-time pad is truly the only unbreakable encryption method, its use is impractical for many modern applications because the system must meet the following conditions:

* The key must be the same size as the message being sent.

* The key must be truly random.

* Keys must be securely shared between the sending and receiving parties.

* If the key is truly random, an xor-based one-time pad is perfectly secure against ciphertext-only cryptanalysis.

* This means an attacker can't compute the plaintext from the ciphertext without knowlege of the key, even via a brute force search of the space of all keys!
